Multi-Tree Seminar Search System

Course: Data Structures and Algorithms
Type: Partner Project
Timeline: Fall, 2024

Project Overview

Developed a multi-tree search system for managing seminar records using various search criteria. The system implements multiple Binary Search Trees (BSTs) for searching by ID, cost, date, and keywords, along with a specialized Bintree for spatial (2D location) searches. This project showcases advanced data structure implementation with a focus on efficient search operations and complex tree traversals. The system supports various search operations including exact matches, range queries, and spatial proximity searches, while maintaining strict tree structure requirements and implementing specific design patterns.

Technical Implementation

  • Data Structures Used:
    • Multiple Binary Search Trees (BSTs) for:
      • ID-based search (no duplicates)
      • Cost-based search (with duplicates)
      • Date-based search (with duplicates)
      • Keyword-based search (with duplicates)
    • Bintree for spatial (2D location) searches
      • Implements Flyweight pattern for empty nodes
      • Uses Composite design pattern for node hierarchy
      • Supports spatial range queries
  • Algorithms Implemented:
    • BST Operations:
      • Inorder traversal for ordered data retrieval
      • Range search algorithms for cost and date queries
      • Modified insertion handling for duplicate keys
    • Bintree Operations:
      • Reverse preorder traversal
      • Spatial search with circular range queries
      • Bounding box intersection calculations
      • Distance calculations for point-radius searches

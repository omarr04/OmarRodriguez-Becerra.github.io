Artist-Song Graph Database System

Course: Data Structures and Algorithms
Type: Partner Project
Timeline: Fall, 2024

Project Overview

Developed a music database system implementing both hash tables and graph data structures to manage relationships between artists and their songs. The system utilizes extensible hash tables for efficient artist and song lookups, along with an adjacency list-based graph structure to represent artist-song relationships. The project features dynamic data structure expansion, string-based hash functions, and graph analysis capabilities including connected components computation using the Union/Find algorithm.

Technical Implementation

  • Data Structures Used:
    • Extensible Hash Tables:
      • One for artist name lookups
      • One for song title lookups
      • Implementation using quadratic probing
      • Dynamic resizing when 50% capacity reached
    • Graph Structure:
      • Adjacency list implementation
      • Nodes representing artists and songs
      • Edges representing recording relationships
      • Support for Union/Find operations
  • Algorithms Implemented:
